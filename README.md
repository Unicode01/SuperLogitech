# SuperLogitech
解决了板载内存不能写脚本的问题（？）

# 食用方法:

1.设置按键F13 - 显示控制台窗口  

2.设置按键F14 - 隐藏控制台窗口  

3.设置按键F15 - Jump脚本  

4.设置按键F16 - AutoClick脚本  

5.设置板载内存模式开启  

6.enjoy
